BACCARAT BOT v2.0 - INSTRUCTIONS

1. Install Tesseract: Run 'tesseract-ocr-w64-setup-5.5.0.20241111.exe' first.
   (IMPORTANT: Keep the default installation path: C:\Program Files\Tesseract-OCR\)
2. Open the Bot: Run 'BaccaratBot.exe'.
3. License: Enter your provided license key (Day, Week, Month, or Year).
4. Goal: Set your 'Stop at Balance' amount in the GUI if you want an auto-stop goal.

TIPS:
- Ensure the game window is fully visible and not covered by other windows.
- Use the Space bar or 'NEXT STEP' button during calibration for easy mapping.
- Do not delete 'config.json', as it contains your calibration data.
