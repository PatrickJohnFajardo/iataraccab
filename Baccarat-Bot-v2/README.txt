Baccarat Automation Bot v2.0 - Portable Distribution

This folder contains pre-configured files and installers for a quick setup.

CONTENTS:
1. BaccaratBot.exe: Compiled version of the bot (No Python required).
2. tesseract-ocr-w64-setup...exe: Installer for the OCR engine.
3. config.json: Configuration file for settings and Supabase keys.
4. start_bot.bat: Launcher for the executable.

QUICK START:
1. Install Tesseract-OCR using the included installer.
2. Edit config.json with your Supabase URL and Key.
3. Run BaccaratBot.exe or start_bot.bat.
4. CALIBRATE the screen once the bot opens.

NOTE: Ensure Windows Display Scaling is set to 100%.
